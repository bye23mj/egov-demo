# 산출물별 필수 컬럼 정의

> `data-output-management` 스킬의 상세 레퍼런스. 본문(SKILL.md)에서 필요할 때만 로드한다(progressive disclosure).
> 각 산출물 xlsx의 `본문` 시트는 아래 컬럼을 헤더로 포함해야 한다.

## 1. 논리데이터모델다이어그램

| 컬럼명 | 설명 |
|---|---|
| 다이어그램ID | 논리 ERD 식별자 |
| 한글DB명 | 논리 DB명 |
| 주제영역 | 데이터 주제영역 |
| ERD표기법 | Mermaid (erDiagram 표기법 고정) |
| ERD코드 | 텍스트 기반 ERD 코드 |
| 포함요소 | 엔터티, 속성, 관계, 식별자 포함 여부 |
| 설명 | 다이어그램 설명 |
| 표준검토상태 | 적합/보완/부적합 |
| 비고 | 보완사항 |
