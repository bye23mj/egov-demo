# 04. 컬럼정의서 — 사용자디렉토리/통합인증

- 대상 DBMS: Oracle 11g / 스키마: KOMSA
- 표준도메인 표기: `도메인분류명 + 데이터타입약어 + 길이` (C=CHAR, V=VARCHAR, N=NUMERIC, D=DATETIME)
- 표준도메인은 공공표준도메인 생성 규칙에 따른 **권고 매핑**이며, 최종 확정은 표준화검토(metadata-agent)에서 수행한다.
- 영문 물리명은 eGov 공통컴포넌트 표준 객체명으로 **원본 유지**.
